# Hotel Management System

## Description

Luxx is an open source hotel management system tailored to digitalized that medium of easing manually booking a hotel. Bringing both hotel host and client through ensuring transparency by means of digitalization.

## Audience

- Hotel host
- Client

## Overview

- Hotel host can analyzed and visualized transactions by means of plotted graphs displaying overview engagement per month. On the other hand approves or decline booking when placed by client after verification.

- Client - stands a chance of experiencing transparency without any third-party negotiation. Rather client has access to book only if the exist individual's user credential upon creating an account. When booking is firstly placed client gets an automatic notification through his/her valid email once used during creating of individual's account. And once booking is verified an automatic email is send with a copy of booking summary as mail.

## Features

- DashBoard

- Visualization of overall engagements

- Content Management Systems
 * create new pages
 * modify existing pages content

- Automatic email notification of one's booking

- Authentication

- AI assistance

- Payment method integration
